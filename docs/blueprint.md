# **App Name**: EldWorkStudio

## Core Features:

- Sticky Navigation Bar: A semi-transparent sticky navigation bar at the top of the page with a backdrop-blur effect.
- Hero Section: A full-height section with the main headline, sub-headline, and CTA button centered vertically and horizontally.
- Text Logo: Display 'EldWorkStudio' as a text logo on the left side of the navigation bar.
- Call to Action Button: Include a 'Get in Touch' button on the right side of the navigation bar and an 'Explore Our Work' button in the Hero Section, both using an iridescent gradient background.
- Responsive Layout: Ensure the layout is perfectly responsive and mobile-first.

## Style Guidelines:

- Main background color: Very dark, near-black (slate-950: #0F172A) for a premium dark-mode theme.
- Primary accent: Vibrant, iridescent gradient that blends bright purple, electric blue, and a hint of emerald green using the Tailwind gradient class: bg-gradient-to-r from-purple-400 via-blue-500 to-emerald-400.
- Font: 'Inter' (sans-serif) for all text. Note: currently only Google Fonts are supported.
- Navigation Bar: Sticky at the top with a semi-transparent background and backdrop-blur effect.
- Hero Section: Full-height (100vh) with content centered vertically and horizontally.
